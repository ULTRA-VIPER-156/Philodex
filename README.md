# PhiloDex

A modern Pokédex application built with React Native and Expo.

Rather than following a single tutorial, this project combines techniques, algorithms, and ideas learned from multiple developers and my Computer Science degree. It serves as both a learning experience and a long-term portfolio project.

## About the Project

PhiloDex started as a way to strengthen my React Native skills outside of university coursework. Over time it has grown into a project where I experiment with application architecture, UI design, responsive layouts, theming, animations, API integration, and overall user experience.

It is still evolving, and I plan to continue improving it as I learn new technologies and development practices.
